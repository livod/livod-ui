导入

```jsx static
import { Button } from "livod-ui";
```

基本使用

```js
<Button>Push Me</Button>
```

danger 类型的 Button

```js
<Button type="danger">Push Me</Button>
```

loading 状态的 Button

```js
<Button loading={true}>Push Me</Button>
```

禁用一个 Button

```js
<Button disabled={true}>Push Me</Button>
```
